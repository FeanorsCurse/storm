Demo

Using links to pass the parameters. Have a look at the requested URLs to see what parameters are passed.

Basics

  • Play a video file
  • Embed preview picture
  • Play immediatelly and loop
  • Choose another skin and its color
  • Preroll videos, e.g. ads
  • Choose player size, auto scale is deactivated
  • Show captions
  • Bigger skin in fullscreen mode

Custom assets

  • Custom preloader and buttonoverlay
  • Custom preloader, buttonoverlay and ending

Debugging

  • Debug mode (Note: "Thunderbolt AS3" or "Firefox + Firebug" required)

Livestreaming

  • Live streaming mode (Note: Flash Media Server required)
